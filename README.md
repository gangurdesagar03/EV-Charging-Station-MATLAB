# Techno-Economic Analysis (TEA) of Renewable Energy Based EV Charging Station

## Overview

This project presents a MATLAB-based techno-economic analysis of a renewable energy powered EV charging station integrating Solar PV, Wind Energy, Battery Storage and Grid connectivity.

## Objectives

* Analyze renewable energy contribution in EV charging infrastructure.
* Reduce dependency on conventional grid power.
* Evaluate economic feasibility of the system.
* Study battery State of Charge (SOC) and grid energy exchange.
* Assess environmental benefits through renewable energy integration.
* Analyze CO₂ emission reduction achieved through renewable energy utilization.

## System Components

* Solar PV System
* Wind Energy System
* Battery Energy Storage System
* Utility Grid
* EV Charging Load

## Analysis Performed

* Energy Generation Analysis
* Battery SOC Analysis
* Grid Energy Exchange Analysis
* Economic Analysis
* CO₂ Emission Reduction Analysis

## Software Used

* MATLAB
* MATLAB Live Script (.mlx)

## Key Results

* Solar Share: 86.03%
* Wind Share: 13.97%
* Grid Share: 0.07%
* LCOE: ₹1.93/kWh
* TNPC: ₹35.16 Lakh
* Payback Period: 3.20 Years
